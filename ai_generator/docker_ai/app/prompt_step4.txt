ユーザが入力したjsonの内容から、犯行の一連の流れを考えて、そのタイムラインをjsonに出力する。

ここでいうuncommonSenseとは、通常起こりえない事象のこと
ここでいうprincipleとは、uncommonSenseを成り立たせる原理もしくは仕組み
ここでいうillusionとは、謎を解かせる人々への錯覚内容

犯人と被害者が必ず登場する。犯人は、itemとtriviaをトリックとして犯行を実行する。使用したトリックは、犯人の行動と結果に何かしらの影響を与える。犯人の行動は、時系列的かつ具体的なものを出力する
犯人以外のことは出力しないこと。
すべて犯人視点で出力すること
主語は省くこと
ステップ数は7個程度までにすること

他の世界観での例を以下に示します.
例1: 
{
    "world": "水族館",
    "item": [
        {
            "name": "水槽",
            "uncommonSense": "水族館の水槽の中で人が移動する",
            "principle": "人は泳ぐごとで移動できる",
            "illusion": "移動性(通路以外の経路で移動することができる)"
        },
        {
            "name": "水",
            "uncommonSense": "水の中でも息ができる",
            "principle": "ビニールシートを水中に持ち込むことで、水槽中の空気排出口の上に空気だまりを作ることができ、息継ぎができる",
            "illusion": "移動性(息が続かず通れない長さの水槽を移動することができる)"
        }
    ],
    "trivia": [
        {
            "name": "エイの毒針は定期的に抜く",
            "uncommonSense": "エイの毒針はすぐに捨てない",
            "principle": "ダストシュートの途中に網をかけることで、特定時間のゴミだけ取得できるようにする",
            "illusion": "能力(取得不可と思われた毒針を入手することができる)"
        }
    ]
}
>> 以上のトリックを用いた「水族館」での「完全犯罪のシナリオ」
・「タッチングプール（魚にさわることができる催し物）」のために、水族館スタッフがエイの毒針を抜く。
・エイの毒針を捨てる場所は分かっているので、そのダストシュートに投げ込まれるタイミングを見計らって、犯人はダストシュート内に網をかけることで、エイの毒針を入手する。（トリック：網を使う。「能力」を錯覚）
・犯人はエイの毒針を、「殺すターゲット」の着るものに仕込む。足止めするために、足に刺さるようにする。
・ターゲットがそれを着る時に、足にでも毒針を受ける。そのためターゲットは医務室に入って、処置をしてもらい、安静にする。
・犯人は計画的に火事を起こして、館内の人をいなくさせる。ターゲットは医務室で足止めされ、どこにも行けない状況になる。
・犯人はビニールシートを使い、息継ぎをしながら水槽内を移動することで、監視カメラにも映らず、ターゲットのいる場所へとたどり着く。（トリック：水槽内移動、ビニールシートトリック。共に「移動性」を錯覚）犯人はターゲットを殺害する。
・犯人は復路（帰り道）でビニールシートの空気を捨てて、ビニールシートを水槽から出し、火事に紛れて燃やすことで、証拠隠滅をする。
・火事鎮圧後に殺しが発覚しても、犯人は現場まで移動不可能だと錯覚されるため、完全犯罪が成立。
